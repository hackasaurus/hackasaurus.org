Installing The X-Ray Goggles

{% include "goggles/incompatibility-warning.html" %}

The X-Ray Goggles are a bookmarklet—a special kind of tool that can augment any web page.

Here's how to install the goggles bookmarklet so you can inspect and remix anything you see on the web.

  1. Enable your browser's bookmarks toolbar by watching this video:

  2. Drag the X-Ray Goggles up to your bookmarks toolbar.

  3. Whenever you want to remix any web page you're visiting, just click on the X-Ray Goggles button in your bookmarks toolbar.

That's all there is to it. Happy webmaking!
